W3

Getting the status of a W3C Standard permits to help a technology to develop quickly and in a good shape. During one year from January 2011, WebID was developed through an Incubator Group (XG) which is the first step in the recommendation redaction process. Incubator Groups are authorized to use W3C infrastructures (such as mailing-list, wiki, bug tracker, ...) and permits to gather experts or all people interested in a technology in one location.

On January 2012, WebID XG will be turned in a Community Group (CG). It's not any kind of evolution but the W3C is deleting the concept of XG to CG.

Node.js

Node.js (Node) is a software system which aims to produce highly scalable application. Programs are written in JavaScript. Node is built ontop of google's V8 engine which is the open source JavaScript engine used for instance in Google Chrome.

To be highly scalable, Node uses event based programmings

Traditional servers such as Apache spawns a new thread (or uses one of a pool) to handle each request. This approach does not provide good performance because of the memory cost of thread or process management. In order to scale well, this kind of server must uses lot of threads.

Node works differently: it uses mainly an event loop that execute your code in a single thread. This loop handle all incoming connections, that's why potentially long operations (such as network or disk I/O, database queries, etc) must be executed asynchronously (with the use of callback) on workers threads.

Many features of JavaScript helps to do event-driven programs effectively (with a minimum of line of code). Moreover, they way that Node is coded allows to do painlessly: there is no thread management nor synchronization functions...

Node is under heavy development;

npm is Node's package manager which permits to install packages (with the additional possibility to add your own c++ wrappers if needed), and even to publish your own package on a centralized platform. On January 7th 2012, there are more than six thousand package available which are covering a wide range of functionalities such as template engines, web pages parsing, certificate management...
